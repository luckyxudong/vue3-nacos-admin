## ADDED Requirements

### Requirement: 面包屑导航
系统 SHALL 在顶部工具栏中提供面包屑导航，显示当前页面的路径层级。

#### Scenario: 面包屑显示
- **WHEN** 用户访问某个页面
- **THEN** 系统应在顶部工具栏显示面包屑导航
- **AND** 面包屑应显示当前页面的完整路径（如：首页 > 概览 > 工作台）
- **AND** 面包屑应使用菜单配置获取菜单标题

#### Scenario: 面包屑生成
- **WHEN** 用户访问页面时
- **THEN** 系统应根据当前路由路径生成面包屑
- **AND** 面包屑应基于菜单配置查找对应的菜单标题
- **AND** 如果路由不在菜单中，应使用路由路径作为标题

#### Scenario: 面包屑点击跳转
- **WHEN** 用户点击面包屑项
- **THEN** 系统应跳转到对应的路由路径
- **AND** 面包屑应更新为新的路径
- **AND** Tab应切换到对应的页面

#### Scenario: 面包屑国际化
- **WHEN** 用户切换语言
- **THEN** 面包屑的文本应更新为对应语言
- **AND** 菜单标题应使用国际化文本
